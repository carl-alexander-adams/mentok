PYTHON_PYO_TARGETS

PYTHON_PYO_TARGETS Overview

PYTHON_PYO_TARGETS is the pattern target used to compile python object files (.pyo files) from python source files (.py files).

PYTHON_PYO_TARGETS Options

<python object target>_SRC
Specifies an override for the default source file. The default value is simply <python object target>.py.

python_pyo

python_pyo Overview

Simple target to trigger the building of PYTHON_PYO_TARGETS pattern targets independent of the main build "target" phase.

Example Makefile


COMPONENT_ROOT=../..
BS_ROOT=$(COMPONENT_ROOT)/../build3

include $(BS_ROOT)/defines.mk


PYTHON_PYO_TARGETS=foo.pyo

foo.pyo_SRC=bar.py
include $(BS_ROOT)/rules.mk
